# ☀️ Solar Assistant RAG Chatbot

An AI-powered Retrieval-Augmented Generation (RAG) chatbot that answers user questions from solar energy documents using Google Gemini, LangChain, and FAISS.

---

## 🚀 Overview

The Solar Assistant RAG Chatbot enables users to interact with solar-related documents through natural language. Instead of searching manually, users can upload PDF documents and ask questions to receive accurate, context-aware responses generated by a Large Language Model (LLM).

---

## ✨ Features

- 📄 Upload and process PDF documents
- 🔍 Semantic search using FAISS vector database
- 🤖 Context-aware question answering with Google Gemini
- 📚 Retrieval-Augmented Generation (RAG)
- 💬 Natural language interaction
- ⚡ Fast document retrieval and response generation

## 📖 How It Works

1. Upload one or more solar-related PDF documents.
2. The application extracts and chunks the text.
3. Text embeddings are generated and stored in a FAISS vector database.
4. Users ask questions in natural language.
5. Relevant document chunks are retrieved.
6. Google Gemini generates an accurate answer using the retrieved context.
